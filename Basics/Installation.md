# Nmap Installation

## Verify Existing Installation

```bash
nmap --version
```

## Install on Ubuntu/Debian

```bash
sudo apt update
sudo apt install nmap
```

## Install on Kali Linux

```bash
sudo apt install nmap
```

## Install on Parrot OS

```bash
sudo apt update
sudo apt install nmap
```

## Verify Installation

```bash
nmap --version
```
