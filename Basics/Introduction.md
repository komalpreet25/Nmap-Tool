# Introduction to Nmap

Nmap (Network Mapper) is an open-source network scanning tool used for host discovery, port scanning, service detection, and security auditing.
Originally created by Gordon Lyon (also known by his pseudonym Fyodor Vaskovich) in 1997, it has become the de facto industry standard for network administrators, penetration testers, and cybersecurity professionals to map out networks and audit their security.

## Features of Nmap

- Host Discovery
- Port Scanning
- Service Detection
- Operating System Detection
- Network Inventory
- Security Auditing

## Why Learn Nmap?

Nmap is one of the most widely used tools in cyber security and penetration testing. It helps security professionals understand network exposure and identify potential security issues.

## Common Use Cases

- Discover active hosts on a network
- Identify open ports
- Detect running services
- Determine operating systems
- Perform security assessments
